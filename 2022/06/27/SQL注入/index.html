堆叠注入局限

就是输入类似1;update user set password = '12345' where id =1这种方式

让执行完查询语句后在执行更新语句，以达到我们的目的

但是很遗憾，这种方式有很大的局限性

主要就是我们常用的ORM框架并不支持同时执行多条sql语句，因此很难实现注入

据了解，这种方式在php语言中可以成功实践。
